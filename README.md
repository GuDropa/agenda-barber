# Agenda Barber

Sistema de agendamento para barbearia com:

- fluxo de cliente para marcar horario
- painel admin para gerenciar agenda, servicos e configuracoes

## Guia para usuarios

Leia o guia didatico completo em:

- [`README-USUARIO.md`](./README-USUARIO.md)

## Executar o projeto

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Rotas principais

- cliente: [http://localhost:3000](http://localhost:3000)
- admin: [http://localhost:3000/admin](http://localhost:3000/admin)
